# 📚 EduPro Online Learning Dashboard

## Internship Project

This project is an interactive data analytics dashboard developed using **Python**, **Pandas**, **Streamlit**, **Matplotlib**, **Seaborn**, and **Plotly**. It analyzes data from an online learning platform and presents meaningful business insights through an interactive dashboard.

---

## 📌 Project Objective

The objective of this project is to analyze the EduPro Online Learning Platform dataset and visualize important insights such as:

- Total Users
- Total Teachers
- Total Courses
- Total Revenue
- Revenue by Course Category
- Revenue by User Gender
- Payment Method Distribution
- Monthly Revenue Trend
- Top 10 Courses by Revenue
- Top 10 Teachers by Revenue

## 🚀 Installation

Clone or download the project.

Install the required libraries:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

Open the project folder in PyCharm or VS Code and run:

```bash
streamlit run app.py
```

If that doesn't work:

```bash
python -m streamlit run app.py
```

The application will open in your browser at:

```
http://localhost:8501
```
